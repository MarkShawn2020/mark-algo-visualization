# 安装指南

## 配置 npm 或者 yarn
本项目完全由`js / ts`编写，所以您需要提前配置好`npm`或者`yarn`，并推荐配置淘宝的镜像源以加快下载包速度，具体可以度娘。

## clone 项目
项目链接：
```shell
git clone git@github.com:MarkShawn2020/mark-algo-visualization.git
```

## 初始化项目
在下载完成后，`cd`到项目根目录，然后
```shell
npm install
```

## 运行项目
```shell
npm run dev
```
将自动启动本地浏览器，打开
`http://localhost:3000`页面。
